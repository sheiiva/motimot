# motimot

Unity word-of-the-day game (inspired by [La palabra del día](https://lapalabradeldia.com/)).

## For contributors

- **Unity project:** open the `Motimot/` folder in Unity (or open this repo root; the project lives in `Motimot/`).
- **Project layout:** see [Motimot/Assets/README.md](Motimot/Assets/README.md) for folder structure, where game logic lives, and the main scene.
- **Game spec and nomenclature:** [docs/GAME_SPECIFICATION.md](docs/GAME_SPECIFICATION.md).
- **Coding style and assemblies:** [docs/CODING_STYLE_AND_ASSEMBLIES.md](docs/CODING_STYLE_AND_ASSEMBLIES.md).
